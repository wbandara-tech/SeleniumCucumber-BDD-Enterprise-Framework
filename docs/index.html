⭐ Enterprise Features

Production-ready capabilities for scalable test automation

🔒

Thread-Safe WebDriver

ThreadLocal implementation ensures each thread gets its own isolated WebDriver instance for reliable parallel execution.

⚡

Parallel Execution

TestNG DataProvider with configurable thread count for faster test execution across multiple scenarios simultaneously.

🌐

Cross-Browser Support

Run tests on Chrome, Firefox, and Edge with configurable browser options, headless mode, and WebDriverManager auto-setup.

🌍

Multi-Environment

Environment-based configuration (QA, UAT) with property files. Switch environments via command line parameters.

📸

Screenshot on Failure

Automatic timestamped screenshot capture on test failure with Allure report attachment and disk storage.

⏳

Smart Wait Utilities

Centralized explicit wait wrapper with configurable timeouts for visibility, clickability, presence, and more.

📝

Centralized Logging

Log4j2 with console and rolling file appenders. Framework-specific logger wrapper for clean, consistent logs.

🔄

Failed Test Rerun

Dedicated FailedTestRunner automatically reruns failed scenarios from the previous execution for reliability.

📊

Data-Driven Testing

Support for Cucumber DataTable, CSV (Apache Commons), and JSON (Jackson) test data sources.

🏷️

Tag-Based Execution

Execute specific test suites with @smoke, @regression, @sanity, @e2e tags and boolean combinations.

🏗️

Page Object Model

Clean POM with abstract BasePage providing 30+ reusable methods. SOLID principles compliance throughout.

🔄

CI/CD Pipeline

GitHub Actions workflow with Java 17 setup, Maven caching, test execution, and artifact uploads.

🚀 Quick Start

Get up and running in minutes

1

Clone the Repository

git clone https://github.com/wbandara-tech/SeleniumCucumber-BDD-Enterprise-Framework.git cd SeleniumCucumber-BDD-Enterprise-Framework
2

Install Dependencies

mvn clean install -DskipTests
3

Run Smoke Tests

mvn clean test -Dcucumber.filter.tags="@smoke"
4

Run All Tests

mvn clean test -Denv=qa -Dbrowser=chrome -Dheadless=true
5

Generate Report

mvn allure:serve
6

Tag Execution

mvn clean test -Dcucumber.filter.tags="@regression" mvn clean test -Dcucumber.filter.tags="@sanity" mvn clean test -Dcucumber.filter.tags="@smoke and @login"
